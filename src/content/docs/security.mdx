# Security Overview

A high-level overview of how ChatIQ protects your data and operation.

---

## Data Handling

- Data is transmitted over TLS.
- Access to customer data is restricted and logged.
- Secrets and API keys should be stored server-side only.

---

## Access Controls

- Team-based access controls in the dashboard.
- API keys are scoped to bots.
- Rate limiting and abuse prevention in place.

---

## Reporting Security Issues

If you believe you have found a security issue, email support@chatiq.io.
