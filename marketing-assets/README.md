# Marketing Assets

This directory contains marketing materials, including video walkthroughs, scripts, and other promotional content.

## Contents

- `loom-walkthrough-script.md` - Script for recording the MVP demo video
- `sample-document-product-faq.md` - Sample FAQ document to upload during the demo
- `loom-links.md` - Links to published Loom videos
- `README.md` - This file

## Usage

1. Review the script before recording
2. Record your walkthrough using Loom or another screen recording tool
3. Add the video link to `loom-links.md`
4. Share the link with early adopters, embed on your website, or include in onboarding emails
